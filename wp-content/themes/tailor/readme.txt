== Installation ==

1. Primary:
 = Login to your wp-admin account and go to Appearance -> Themes.
 = Select "Install" tab and click on the "Upload" link.
 = Select "tailor.zip" and click on "Install Now" button.
 = In case of errors, use an alternate method.
 
2. Alternate:
 = Unzip the template file (tailor.zip) that you have downloaded.
 = Upload the entire folder (tailor) to your server via FTP and place it in the /wp-content/themes/ folder.
 = Do not change the directory name.
 = The template files should be there now: /wp-content/themes/tailor/index.php (example).
 
3. Log into your WP admin panel and click on "Appearance". Go to "Themes" tab.

4. Now click on "tailor" to activate the theme.

5. Theme Customizer Options:	

	a) Basic Settings:
           You can manage here site logo, favicon, blog title, footer text, and copyright text 	

	b) Home Page Settings:
		i)   Slider Section : You can set title, sub-title, background image, fa-fa class button text and title.
		ii)  About section : You can set title, sub-title, image, and description 
        	iii) Blog Section : you can select category of posts and add title, description of posts slider.
                
   	c) Social Settings:
            You can add here social media links.
